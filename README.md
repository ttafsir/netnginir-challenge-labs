# netnginir-challenge-labs
A collection of activities and challenges labs to be used with [netnginir-ansible-lab](https://github.com/ttafsir/netnginir-ansible-lab) to practice network automation with Ansible.

## Labs are grouped under the following objectives 
1. [Adhoc commands](./challenge_labs/Adhoc_commands.md)
2. [Exploring The Inventory](./challenge_labs/Exploring_The_Inventory.md)
3. [Exploring Network Modules](./challenge_labs/Exploring_Network_Modules.md)
4. [Plays And Playbooks](./challenge_labs/Plays_And_Playbooks.md)
5. [Working With Templates](./challenge_labs/Working_With_Templates.md)
6. [Variables And Facts](./challenge_labs/Variables_And_Facts.md)
