# netnginir-challenge-labs
A collection of activities and challenges labs to be used with [netnginir-ansible-lab](https://github.com/ttafsir/netnginir-ansible-lab) to practice network automation with Ansible.

## Labs are grouped under the following objectives 
1. [Adhoc commands](./challenge_labs/1_Adhoc_commands.md)
2. [Exploring The Inventory](./challenge_labs/2_Exploring_The_Inventory.md)
3. [Exploring Network Modules](./challenge_labs/3_Exploring_Network_Modules.md)
4. [Plays And Playbooks](./challenge_labs/4_Plays_And_Playbooks.md)
5. [Working With Templates](./challenge_labs/5_Working_With_Templates.md)
6. [Variables And Facts](./challenge_labs/6_Variables_And_Facts.md)
